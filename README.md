# 2D Graphics Editor in C

## Description

A menu-driven 2D Graphics Editor developed in C using a character-based canvas (2D array). The project allows users to draw, display, modify, and delete graphical objects such as lines, rectangles, triangles, and circles.

## Features

* Display Canvas
* Draw Horizontal Line
* Draw Vertical Line
* Draw Rectangle
* Draw Triangle
* Draw Circle
* Clear Canvas
* Save Canvas to File
* Load Canvas from File
* Draw Text
* Undo Last Action
* Delete Rectangle
* Modify Rectangle

## Technologies Used

* C Programming Language
* 2D Arrays
* Functions
* File Handling

## Project Structure

* `main.c` – Menu-driven user interface
* `graphics.c` – Graphics functions implementation
* `graphics.h` – Function declarations and constants

## Compile

```bash
gcc main.c graphics.c -o editor
```

## Run

```bash
./editor
```

or on Windows:

```bash
editor.exe
```
